Hermes

Summary

This is a minimal wearable computer that lets you take the power of a workstation with you. By using a remote connection, you can make use of all of your home computer's applications while on the road.

Description

The Hermes line of wearable computers is a complete offering for remote computing. Minimizing the requirements of a wearable computer, it is both economical and powerful. It uses your home desktop computer as a server and the Hermes acts as a terminal -- painting your display with the images from the server. The Hermes also contains an optional Global Positioning System to keep track of where you are.
