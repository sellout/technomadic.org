OS X Scripts

All the little scripts I’ve written for OS X.

Compress This (download)

Put this in Finder’s toolbar and you can use it to make a compressed archive of the current directory.

MailImages (download)

I seem to remember that in 10.1, Mail.app would show the image from your AddressBook for whomever you received mail from. That doesn’t happen in Jaguar. This little script (meant to be run periodically, probably from cron) gives you back that functionality.

Terminal Here (download)

If you put this in Finder’s toolbar, it’ll open a Terminal with the directory set to the one the Finder window is showing.
