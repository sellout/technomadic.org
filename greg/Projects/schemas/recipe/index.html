Recipe Schema

This schema is designed to hold cooking recipes.

recipe.xsd
The recipe schema file.
recipe.css
A basic stylesheet for the recipes.
recipe.xsl
An XSLT file for transforming your recipes into XHTML.
recipe.xml
An empty template to create your own recipes in.
