Screenplay Schema

This schema is designed to hold screenplays.

screenplay.xsd
The screenplay schema file.
screenplay.css
A basic stylesheet for the screenplays.
screenplay-typewriter.css
A stylesheet for making the screenplay look like the traditional typewriter style.
screenplay-xhtml.xsl
An XSLT file for transforming your screenplays into XHTML.
screenplay-typewriter.xsl
An XSLT file for transforming your screenplays into a plain text file in the traditional typewriter style.
screenplay.xml
An empty template to create your own screenplays in.
